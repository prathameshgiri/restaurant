# Spice Garden Restaurant

A full-featured, responsive restaurant website and management project created as a college project for **Aditi Suryawanshi**.

**Designed and Developed by:** [Prathamesh Giri](https://prathameshgiri.in)

## ✨ Features

*   **User Interface:** Beautiful and responsive UI for customers to view the menu, read the story, and submit reviews.
*   **Authentication:** Distinct login portals and role-based access for Users and Admins.
*   **Order Management:** Cart system, checkout process, and order tracking for users.
*   **Admin Dashboard:** Complete management panel for admins to oversee orders, incoming reservations, messages, and menu items.
*   **Dynamic Data:** Stores changes (orders, users, menu items) seamlessly using local JSON file storage.

## 🚀 How to Run the Project

Follow these steps to get the project up and running locally on your machine:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation

1.  Open your terminal or command prompt.
2.  Navigate to the project directory:
    ```bash
    cd "d:\MDM.Clg.Pro\Aditi Suryawanshi\Restaurant Final"
    ```
3.  Install the required dependencies:
    ```bash
    npm install
    ```

### Starting the Server

Start the project by running:
```bash
npm start
```
Alternatively, you can run `node server.js`.

The server will start running on port `3001`.
